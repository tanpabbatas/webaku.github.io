A: HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan tampilan konten di web.
Q: Apa perbedaan antara HTML dan CSS?
A: HTML digunakan untuk struktur konten (seperti teks, gambar, link), sementara CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan gaya dari konten HTML.
Q: Apakah JavaScript?
